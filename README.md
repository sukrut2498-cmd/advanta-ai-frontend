## 🏛️ Frontend UI Architecture Specification
The user interface is powered by an asynchronous **Streamlit** framework optimized to manage user session states natively across multiple application context panels.

### Modular Interface Structure:
* **Dashboard Gateway:** Executive KPI card row detailing generation statistics, compliance trends, and operational system alerts.
* **Campaign Studio:** Configuration forms built with multi-part payload packaging, a 0-byte streaming file guard block, and real-time Instagram/Web composition live previews.
* **Compliance Center:** Dynamic progress gauges tracking specific tone annotations and brand violation notices.
* **Scheduler:** Interactive date/time pickers tied to a downstream active pipeline dispatch ledger tracking dataframes.
* **Audit Trail:** An expander-based, timestamped revision timeline tracking session state history cache structures.
* **Analytics:** Real-time visualization using native line and bar charts showing historical brand health indexes.

---

## 🌓 Adaptive Multi-Theme Compliant Styling
The frontend UI code entirely avoids hardcoded hex values, utilizing **Streamlit CSS Native Theme Variables** to render elegantly in both Light and Dark themes.

* **Layout Matching:** Binds background containers to `var(--background-color)` and cards to `var(--secondary-background-color)`.
* **Contrast Preservations:** Dynamically scales typography states using `var(--text-color)`, preventing text washing out when the client alters global browser theme settings.
